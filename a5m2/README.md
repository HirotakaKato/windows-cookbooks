# はじめに

これは Windows に
[A5:SQL Mk-2](http://www.wind.sannet.ne.jp/m_matsu/developer/a5m2/)
をインストールする Cookbook です。

# 処理内容

1. 64bit または 32bit 用の zip ファイルをダウンロードします。
2. C:\opt\a5m2 に zip ファイルを展開します。
3. スタートメニューにショートカットを作成します。
